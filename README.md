Criação completa de uma aplicação web simples desenvolvida como atividade acadêmica, implementando um sistema simples de tabuada. A aplicação inclui:<br/>
•	Validações de entrada: garante que o usuário digite apenas números inteiros entre 1 e 10.<br/>
•	Interface intuitiva e responsiva: layout limpo, com design simples e agradável, adaptável a diferentes tamanhos de tela.<br/>
•	Funcionalidades práticas: botão para calcular a tabuada do número informado e botão “Limpar” para apagar todo o conteúdo de forma rápida.<br/>
•	Código estruturado e moderno: uso de HTML, CSS e JavaScript, com estilo inspirado em interfaces minimalistas, incluindo efeitos visuais leves e feedbacks de interação.<br/>
O resultado é uma aplicação didática e interativa, que permite aos usuários aprender e praticar a tabuada de forma visual e direta, tornando a experiência de uso simples e agradável.
